What is Morse Code?

Morse code is a way to send messages using a series of short and long sounds or flashes of light.
Think of it like a secret language that uses "dots" and "dashes" instead of letters.
A "dot" is a quick, short sound, like a tap on a table, and a "dash" is a longer sound, like a knock that lasts a bit longer.
Each letter of the alphabet has its own special pattern of dots and dashes.
For example, the letter "A" is a dot followed by a dash (.-), and the letter "B" is a dash followed by three dots (-...).
